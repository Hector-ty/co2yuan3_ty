# RAGFlow 中选择 Embedding 模型的位置

## 📍 选择位置

### 方法 1：创建知识库时选择（推荐）

1. **登录 RAGFlow**
   - 访问：`http://localhost:8082`
   - 登录您的账号

2. **创建知识库**
   - 点击左侧菜单 "知识库" 或 "数据集"
   - 点击 "新建" 或 "+" 按钮

3. **在创建表单中选择 Embedding 模型**
   - 在创建知识库的表单中，会有一个 **"嵌入模型"** 或 **"Embedding Model"** 的下拉选择框
   - 点击下拉框，会显示所有可用的 Embedding 模型

4. **选择 Embedding 模型**
   - **中文推荐**：
     - 在 **GiteeAI** 工厂中搜索：`bge-small-zh-v1.5`（注意：小写，没有 `BAAI/` 前缀）
     - 或使用 **Builtin** 工厂中的：`BAAI/bge-m3`（多语言模型，支持中文）
   - **英文推荐**：
     - 在 **Builtin** 工厂中选择：`BAAI/bge-small-en-v1.5`
   - **注意**：模型名称格式可能因工厂而异，请根据实际显示的名称选择

### 方法 2：在系统设置中配置默认 Embedding 模型

1. **进入系统设置**
   - 点击右上角用户头像
   - 选择 "系统设置" → "默认模型" 或 "Default Models"

2. **设置默认 Embedding 模型**
   - 找到 "Embedding Model" 或 "嵌入模型" 选项
   - 选择 `bge-small-zh-v1.5`（GiteeAI 工厂）或 `BAAI/bge-m3`（Builtin 工厂）
   - 保存设置

3. **创建知识库时自动使用**
   - 创建新知识库时，会自动使用您设置的默认 Embedding 模型

### 方法 3：在模型管理中添加 Embedding 模型

如果需要使用其他 Embedding 模型（如 Ollama 的）：

1. **进入模型管理**
   - 系统设置 → "LLM 模型" 或 "模型管理"
   - 点击 "添加模型"

2. **添加 Embedding 模型**
   - **模型名称**：例如 `nomic-embed-text`
   - **模型类型**：选择 `Embedding` 或 `TEXT EMBEDDING`
   - **LLM Factory**：选择 `Ollama`（如果使用 Ollama）
   - **Base URL**：`http://host.docker.internal:11434`
   - **API Key**：留空

3. **在创建知识库时选择**
   - 创建知识库时，下拉框中会显示您添加的模型

## 🎯 推荐配置

### 对于中文项目

**推荐使用以下模型之一：**

1. **GiteeAI 工厂**：
   - **模型名称**：`bge-small-zh-v1.5`（注意：小写，没有 `BAAI/` 前缀）
   - **优点**：
     - ✅ 专为中文优化
     - ✅ 性能好
     - ✅ 无需额外配置

2. **Builtin 工厂**：
   - **模型名称**：`BAAI/bge-m3`
   - **优点**：
     - ✅ 多语言支持（包括中文）
     - ✅ 支持更长文本（8k tokens）
     - ✅ 无需额外配置

### 对于英文项目

**推荐使用：**
- **模型名称**：`BAAI/bge-base-en-v1.5`
- 或 `BAAI/bge-large-en-v1.5`（性能更好但更大）

## ⚠️ 重要提示

1. **模型一旦选择，不能轻易更改**
   - 知识库创建后，如果已经上传了文档并生成了向量
   - 不能直接更改 Embedding 模型
   - 需要删除所有文档块（chunks）后才能更改

2. **同一知识库必须使用相同的 Embedding 模型**
   - 所有文档必须使用同一个 Embedding 模型处理
   - 否则向量空间不一致，无法正确检索

3. **内置模型 vs Ollama 模型**
   - **内置模型**：性能更好，推荐使用
   - **Ollama 模型**：需要额外安装，适合本地部署场景

## 📝 操作步骤总结

### 最简单的步骤（使用内置模型）：

1. 登录 RAGFlow → `http://localhost:8082`
2. 点击 "知识库" → "新建"
3. 填写知识库名称
4. 在 "嵌入模型" 下拉框中选择：
   - **GiteeAI** 工厂 → `bge-small-zh-v1.5`（中文推荐）
   - 或 **Builtin** 工厂 → `BAAI/bge-m3`（多语言）
5. 点击 "创建"
6. 上传文档开始使用

## 🔍 如果找不到 Embedding 模型选项

可能的原因：
1. 界面版本不同，可能在 "高级设置" 或 "更多选项" 中
2. 需要先在系统设置中添加 Embedding 模型
3. 检查是否有权限创建知识库

## 💡 提示

- 如果主要处理中文文档，推荐使用：
  - **GiteeAI** 工厂的 `bge-small-zh-v1.5`（专为中文优化）
  - 或 **Builtin** 工厂的 `BAAI/bge-m3`（多语言支持）
- 这些模型无需额外配置，开箱即用
- **重要**：搜索时使用 `bge-small-zh-v1.5`（小写，无 `BAAI/` 前缀），不要搜索 `BAAI/bge-small-zh-v1.5`
