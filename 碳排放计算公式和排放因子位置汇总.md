# 碳排放计算公式和排放因子位置汇总

本文档汇总了项目中所有涉及碳排放计算公式和排放因子的代码位置。

**重要更新**: 以下排放因子已从系统中删除：
- **固体燃料（已弃用）**: 炼焦煤(cokingCoal)、型煤(briquettes)、焦炭(coke)、其它焦化产品(otherCokingProducts)
- **液体燃料（已弃用）**: 原油(crudeOil)、石脑油(naphtha)、沥青(asphalt)、润滑油(lubricants)、石油焦(petroleumCoke)、石化原料油(petrochemicalFeedstock)、其它油品(otherOils)
- **气体燃料（已弃用）**: 炼厂干气(refineryGas)
- **移动源（已弃用）**: 整个移动源类别已弃用，不再进行计算

这些字段已从所有代码、数据库模型和文档中移除。计算引擎已更新以兼容旧数据（自动跳过这些已弃用的字段）。

**数据库清理**: 如需从MongoDB数据库中删除这些已弃用的排放因子记录，可以调用以下API：
- **API端点**: `POST /api/emission-factors/delete-deprecated`
- **权限要求**: 管理员权限
- **功能**: 批量删除数据库中已弃用的排放因子记录
  - 固体燃料: `cokingCoal`, `briquettes`, `coke`, `otherCokingProducts`
  - 液体燃料: `crudeOil`, `naphtha`, `asphalt`, `lubricants`, `petroleumCoke`, `petrochemicalFeedstock`, `otherOils`

## 一、核心计算公式位置

### 1. 后端计算引擎（主要计算公式实现）

**文件路径**: `server/utils/calculationEngine.js`

**关键函数**: `calculateEmissions(activityData)`

**计算公式详情**:

#### 1.1 固体燃料排放计算
- **位置**: 第64-81行
- **公式**: `排放量 = 燃料消耗量(t) × 排放因子(tCO₂/t)`
- **代码**: 
  ```javascript
  breakdown.fossilFuels += (fossilFuels.solid[fuel] || 0) * factor;
  ```

#### 1.2 液体燃料排放计算
- **位置**: 第82-100行
- **公式**: `排放量 = 燃料消耗量(t) × 排放因子(tCO₂/t)`
- **代码**: 
  ```javascript
  breakdown.fossilFuels += (fossilFuels.liquid[fuel] || 0) * factor;
  ```

#### 1.3 气体燃料排放计算（含单位换算）
- **位置**: 第101-118行
- **公式**: `排放量 = (燃料消耗量(m³) / 10000) × 排放因子(tCO₂/万Nm³)`
- **代码**: 
  ```javascript
  breakdown.fossilFuels += ((fossilFuels.gas[fuel] || 0) / 10000) * factor;
  ```
- **单位换算说明**: 输入单位为 m³，需要除以 10000 转换为万Nm³

#### 1.4 移动源排放计算（已弃用）
- **状态**: **已弃用** - 移动源排放因子已被弃用，不再进行计算
- **说明**: 如果旧数据中存在mobileSources，将被忽略，不再计入排放总量

#### 1.5 逸散排放计算
- **位置**: 第125-188行
- **分类**: 
  - **空调系统逸散排放**: HCFC-22, HFC-32, HFC-125, HFC-134a, HFC-143a, HFC-227a, HFC-245fa
  - **灭火系统逸散排放**: CO2, HFC-227ea
- **公式**: 
  - **空调系统逸散排放**: `空调系统逸散排放 = Σ(用户输入值 × 排放因子)`（仅当输入值 ≥ 阈值时计入）
  - **灭火系统逸散排放**: `灭火系统逸散排放 = Σ(用户输入值 × 排放因子)`（仅当输入值 ≥ 阈值时计入）
  - **总逸散排放**: `总逸散排放 = 空调系统逸散排放 + 灭火系统逸散排放`
- **阈值逻辑**: 当用户输入值小于以下阈值时，该部分逸散排放取零
  - HCFC-22: < 1.29
  - HFC-32: < 3.27
  - HFC-125: < 0.67
  - HFC-143a: < 0.43
  - HFC-245fa: < 2.62
  - CO2: < 2.52
  - HFC-227ea: < 23.33
- **代码**: 
  ```javascript
  // 空调系统逸散排放
  airConditioningGases.forEach(gas => {
    const inputValue = fugitiveEmissions.airConditioning[gas] || 0;
    const threshold = fugitiveThresholds[gas];
    if (threshold !== undefined && inputValue < threshold) {
      return; // 跳过该气体
    }
    airConditioningTotal += inputValue * factor;
  });
  
  // 灭火系统逸散排放
  fireSuppressionGases.forEach(gas => {
    const inputValue = fugitiveEmissions.fireSuppression[gas] || 0;
    const threshold = fugitiveThresholds[gas];
    if (threshold !== undefined && inputValue < threshold) {
      return; // 跳过该气体
    }
    fireSuppressionTotal += inputValue * factor;
  });
  
  breakdown.fugitiveEmissions = breakdown.airConditioning + breakdown.fireSuppression;
  ```
- **说明**: 逸散排放因子从MongoDB数据库中读取（category为"fugitive"），不在默认配置文件中定义

#### 1.6 间接排放计算（含单位换算）
- **位置**: 第190-204行
- **电力排放公式**: `排放量 = 净外购电量(10⁴kWh) × 10 × 排放因子(tCO₂/MWh)`
- **热力排放公式**: `排放量 = 净外购热力(GJ) × 排放因子(tCO₂e/GJ)`
- **代码**: 
  ```javascript
  // 电力
  breakdown.electricity = (indirectEmissions.purchasedElectricity || 0) * 10 * dbFactors.indirect.electricity;
  // 热力
  breakdown.heat = (indirectEmissions.purchasedHeat || 0) * dbFactors.indirect.heat;
  ```
- **单位换算说明**: 
  - 电力：10⁴kWh = 10 MWh，所以乘以 10
  - 热力：排放因子单位为 tCO₂e/GJ，直接使用（不需要除以1000）

#### 1.7 汇总计算
- **位置**: 第206-210行
- **公式**: 
  - **直接排放总量**: `直接排放总量(tCO₂) = 化石燃料燃烧排放量(tCO₂) + 逸散排放总量(tCO₂)`
  - **间接排放总量**: `间接排放总量(tCO₂) = 外购电力排放量(tCO₂) + 外购热力排放量(tCO₂)`
  - **碳排放总量**: `碳排放总量(tCO₂) = 直接排放总量(tCO₂) + 间接排放总量(tCO₂)`
- **代码**: 
  ```javascript
  // 直接排放总值 = 燃料燃烧总值 + 逸散排放总值
  const totalDirect = breakdown.fossilFuels + breakdown.fugitiveEmissions;
  const totalIndirect = breakdown.electricity + breakdown.heat;
  const totalEmissions = totalDirect + totalIndirect;
  ```

#### 1.8 强度指标计算
- **位置**: 第212-218行
- **公式**: 
  ```javascript
  // 单位建筑面积碳排放强度
  const emissionIntensityByArea = intensityMetrics?.buildingArea > 0 
    ? totalEmissions / intensityMetrics.buildingArea 
    : 0;
  // 人均碳排放强度
  const emissionIntensityByPerson = intensityMetrics?.personnelCount > 0 
    ? totalEmissions / intensityMetrics.personnelCount 
    : 0;
  ```

### 2. 前端实时计算（表单预览）

**文件路径**: `client/src/components/DataEntryForm.jsx`

**关键函数**: `calculateEmission(field)` (第167-209行)

**计算公式**:
- **固体/液体燃料**: `排放量 = 消耗量 × 排放因子`
- **气体燃料**: `排放量 = (消耗量 / 10000) × 排放因子`
- **逸散排放**: `排放量 = 用户输入值 × 排放因子`（仅当输入值 ≥ 阈值时计入）
  - **阈值**: HCFC-22(1.29), HFC-32(3.27), HFC-125(0.67), HFC-143a(0.43), HFC-245fa(2.62), CO2(2.52), HFC-227ea(23.33)
- **移动源**: **已弃用** - 不再进行计算
- **外购电力**: `排放量 = 电量 × 10 × 排放因子`
- **外购热力**: `排放量 = 热力 × 排放因子`

**代码片段**:
```javascript
if (category === 'fossilFuels' && emissionFactors[type] && emissionFactors[type][fuel]) {
    const factor = emissionFactors[type][fuel];
    emission = type === 'gas' ? (value / 10000) * factor : value * factor;
} else if (category === 'fugitiveEmissions' && emissionFactors.fugitive) {
    const gasName = fuel;
    const factor = emissionFactors.fugitive[gasName];
    if (factor !== undefined && factor !== null) {
      const threshold = fugitiveThresholds[gasName];
      if (threshold !== undefined && value < threshold) {
        return 0; // 小于阈值时取零
      }
      emission = value * factor;
    }
} else if (category === 'mobileSources' && emissionFactors.mobile?.[type]?.[fuel]) {
    const factor = emissionFactors.mobile[type][fuel];
    emission = (value * factor) / 1000;
} else if (category === 'indirectEmissions' && emissionFactors.indirect) {
    if (type === 'purchasedElectricity') emission = value * 10 * emissionFactors.indirect.electricity;
    else if (type === 'purchasedHeat') emission = value * emissionFactors.indirect.heat;
}
```
