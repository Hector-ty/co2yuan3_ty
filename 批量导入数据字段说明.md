# 批量导入数据模板字段说明

## 必填字段

以下字段在批量导入时**必须**提供：

| Excel/CSV列名示例 | 数据库字段路径 | 说明 | 数据类型 |
|------------------|--------------|------|---------|
| 组织机构代码 | `regionCode` | 行政区划代码 | 字符串 |
| 年份 | `year` | 数据年份（也可从文件名提取，如：呼和浩特2024年1月.xls） | 数字 |

## 多级表头支持

系统支持多级表头（如"建筑面积(平方米)_当年"），格式为：`第一行表头_第二行表头`

例如：
- 第一行：`建筑面积(平方米)`
- 第二行：`当年`
- 合并后：`建筑面积(平方米)_当年`

## 单位转换说明

系统会自动进行以下单位转换：

1. **电消费量**：千瓦时 → 10⁴ kWh（除以10000）
2. **天然气消费量**：立方米 → 10⁴ m³（除以10000）
3. **汽油消费量**：升 → 千克（乘以0.75，汽油密度约0.75 kg/L）
4. **柴油消费量**：升 → 千克（乘以0.85，柴油密度约0.85 kg/L）
5. **液化石油气**：千克 → 吨（除以1000）

## 示例映射模板

参考 `呼和浩特数据映射模板.json` 文件，示例映射如下：

```json
{
  "templateName": "呼和浩特2024年数据导入模板",
  "mappings": {
    "组织机构代码": "regionCode",
    "建筑面积(平方米)_当年": "activityData.intensityMetrics.buildingArea",
    "用能人数(人)_当年": "activityData.intensityMetrics.personnelCount",
    "电消费量(千瓦时)_当年": "activityData.indirectEmissions.purchasedElectricity",
    "煤炭消费量(吨)_当年": "activityData.fossilFuels.solid.bituminousCoal",
    "天然气消费量(立方米)_当年": "activityData.fossilFuels.gas.naturalGas",
    "汽油消费量(升)_当年": "activityData.fossilFuels.liquid.gasoline",
    "柴油消费量(升)_当年": "activityData.fossilFuels.liquid.diesel",
    "热力消费量(吉焦)_当年": "activityData.indirectEmissions.purchasedHeat",
    "液化石油气消费量(千克)_当年": "activityData.fossilFuels.liquid.lpg"
  }
}
```

## 注意事项

1. **必填字段**：`regionCode`（组织机构代码）和 `year`（年份）必须提供
2. **年份提取**：如果文件名包含年份（如"呼和浩特2024年1月.xls"），系统会自动提取年份
3. **数据验证**：所有数值字段必须是有效数字，否则会报错
4. **已弃用字段**：以下字段已弃用，请勿使用：
   - `activityData.fossilFuels.solid.cokingCoal`（焦煤）
   - `activityData.fossilFuels.solid.briquettes`（型煤）
   - `activityData.fossilFuels.solid.coke`（焦炭）
   - `activityData.fossilFuels.solid.otherCokingProducts`（其他焦化产品）
   - `activityData.fossilFuels.liquid.crudeOil`（原油）
   - `activityData.fossilFuels.liquid.naphtha`（石脑油）
   - `activityData.fossilFuels.liquid.asphalt`（沥青）
   - `activityData.fossilFuels.liquid.lubricants`（润滑油）
   - `activityData.fossilFuels.liquid.petroleumCoke`（石油焦）
   - `activityData.fossilFuels.liquid.petrochemicalFeedstock`（石化原料）
   - `activityData.fossilFuels.liquid.otherOils`（其他油品）
   - `activityData.fossilFuels.gas.refineryGas`（炼厂气）
   - `activityData.mobileSources.*`（移动源相关字段，请使用 `activityData.fossilFuels.liquid.*` 或 `activityData.mobileSourcesDetail.*`）

## 文件格式要求

- 支持格式：`.csv`, `.xlsx`, `.xls`
- 表头：支持单级表头和多级表头
- 编码：CSV文件建议使用UTF-8编码
