# EmissionFactor 文件夹集成说明

## 概述

`EmissionFactor` 文件夹包含了一个新的排放因子管理系统，使用 TypeScript + tRPC + MySQL 技术栈。为了将其集成到现有项目（JavaScript + Express + MongoDB）中，我们采用了以下策略：

## 集成方案

### 1. 数据模型扩展

已扩展 `server/models/EmissionFactor.js` 以支持新系统的详细分类：

- **新增字段**：
  - `fuelTypeCn` / `fuelTypeEn`: 化石燃料的中英文名称
  - `gasNameCn` / `gasNameEn`: 逸散排放气体的中英文名称
  - `gwpValue`: GWP值（用于逸散排放）
  - `emissionTypeCn` / `emissionTypeEn`: 间接排放类型的中英文名称
  - `emissionFactor`: 字符串格式的排放因子值（支持 "100%" 这样的值）
  - `gwp`: GWP值（用于化石燃料）
  - `remarks`: 备注（用于间接排放）
  - `fugitiveCategory`: 逸散排放的子分类（airConditioning / fireSuppression）

- **新增分类**：
  - `fossil`: 化石燃料
  - `fugitive`: 逸散排放

### 2. 数据迁移

已创建迁移脚本 `server/utils/migrateEmissionFactors.js`，可以将新系统的种子数据导入到现有 MongoDB 数据库。

#### 使用方法

1. **通过 API 调用迁移**（推荐）：
   ```bash
   POST /api/emission-factors/migrate
   ```
   需要管理员权限（admin role）

2. **直接运行迁移脚本**：
   ```javascript
   const { migrateNewSystemData } = require('./server/utils/migrateEmissionFactors');
   await migrateNewSystemData();
   ```

### 3. 新系统数据分类

新系统将排放因子分为三类：

#### 化石燃料（Fossil Fuels）
- 无烟煤、烟煤、褐煤
- 焦炉煤气、管道煤气、天然气
- 汽油、柴油、燃料油、煤油
- 液化石油气、液化天然气

#### 逸散排放（Fugitive Emissions）
- **空调系统**：HCFC-22, HFC-32, HFC-125, HFC-134a, HFC-143a, HFC-227a, HFC-245fa
- **灭火系统**：CO₂灭火器, FM200

#### 间接排放（Indirect Emissions）
- 外购电力
- 外购热力

## 前端集成建议

新系统的前端组件使用 TypeScript + Tailwind CSS，而现有项目使用 JavaScript + Material-UI。可以选择以下方案：

### 方案 A：保持现有前端（推荐）
- 继续使用现有的 `EmissionFactorManagementPage.jsx`
- 通过 API 获取新迁移的数据
- 在现有页面中显示新的分类

### 方案 B：逐步迁移前端组件
- 将新系统的 TypeScript 组件转换为 JavaScript
- 将 Tailwind CSS 样式转换为 Material-UI 组件
- 集成到现有页面中

## 数据兼容性

迁移脚本会自动处理数据格式转换：

1. **化石燃料**：根据燃料类型自动映射到 `solid`、`liquid` 或 `gas` 分类
2. **逸散排放**：保持 `fugitive` 分类，并记录子分类（空调系统/灭火系统）
3. **间接排放**：保持 `indirect` 分类

## 注意事项

1. **数据去重**：迁移脚本使用 `upsert` 操作，不会重复插入已存在的数据
2. **向后兼容**：现有系统的数据结构和 API 保持不变
3. **字段可选**：新字段都是可选的，不会影响现有功能

## 使用示例

### 迁移新系统数据

```javascript
// 在浏览器控制台或 Postman 中
fetch('/api/emission-factors/migrate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  }
})
.then(res => res.json())
.then(data => console.log(data));
```

### 查询迁移后的数据

```javascript
// 获取所有排放因子（包括新迁移的数据）
fetch('/api/emission-factors', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
})
.then(res => res.json())
.then(data => {
  // 筛选化石燃料
  const fossilFuels = data.filter(f => f.category === 'fossil');
  // 筛选逸散排放
  const fugitiveEmissions = data.filter(f => f.category === 'fugitive');
  // 筛选间接排放
  const indirectEmissions = data.filter(f => f.category === 'indirect');
});
```
